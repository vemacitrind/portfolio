# Arch Linux–Inspired Portfolio

A unique, desktop-themed portfolio website inspired by the aesthetics and minimalism of Arch Linux. The interface replicates a Linux desktop environment, including a functional terminal that accepts predefined commands to showcase information about the developer.

## Features

- **Desktop UI Simulation**: Styled after a typical Arch Linux environment.
- **Interactive Terminal**: Accepts predefined commands such as `about`, `projects`, `contact`, etc.
- **Firefox Integration**: Includes a mock Firefox browser window to enhance the realism of the desktop environment.
- **Modular Design**: Built with reusable components for scalability and maintainability.
- **Lightweight & Performant**: Optimized for fast load times and minimal resource usage.

## Technologies Used

- **React** (Frontend framework)
- **CSS** (Custom styling, no utility framework)
- **React-RND** (Resizable and draggable windows)
- **Custom Terminal Logic** (JavaScript-based command parser)
- **Icons / Assets**: Linux desktop–style icons and UI elements
- **Create React App (CRA)** (Project scaffolding and build tool)
